# Todoist Operating Rules

## Organization Philosophy
- **Time vs Type:** We use **Projects** for Time (This Week, Next Month) and **Labels** for Type (`type-home`, `type-work`, etc.).
- **No Orphans:** Every task must have a Section. If unsure, use "Inbox" section.
- **Backburners:** Use "Personal Backburner" (Inbox) or "🔥➡️❄️ Backburner" (#instacart) for seasonal/deferred items.

## Ownership & Attention
| Label | Meaning |
|-------|---------|
| `magi` | I own/do this task |
| `needs-magi` | **Alert:** Eric wants me to look at this (I scan for this) |
| `cray` / `marvin` | Other bots |
| `needs-eric` | Bot needs Eric's input |

## 1:1 & Relationships
- **Project: 1:1 Topics** - Sections for work contacts (Puvi, Nikita, etc.)
- **Project: Relationships** - Sections for personal (Mom, Nadja)
- **Workflow:** Add topics to sections -> Discuss -> Check off.

## Automated Routing
- **Magi created:** Defaults to Personal/Home context
- **Cray created:** Defaults to Work/#instacart context
- **Marvin created:** Defaults to Homelab/Infra context
