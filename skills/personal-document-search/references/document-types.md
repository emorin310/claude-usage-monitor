# Personal Document Types and Search Patterns

This reference defines the categories, search patterns, and file types for personal documents commonly stored in Google Drive and BigStore.

## Document Categories

### Identity Documents
**Purpose:** Government-issued identification and citizenship documents

**Document Types:**
- **Social Insurance Number (SIN)** 
  - Search terms: `sin`, `social insurance`, `social insurance number`
  - File types: PDF, JPG, PNG
  - Common names: `SIN-card.pdf`, `social-insurance.jpg`

- **Driver's License**
  - Search terms: `license`, `licence`, `driver`, `driving`  
  - File types: PDF, JPG, PNG
  - Common names: `drivers-license.pdf`, `ontario-license.jpg`

- **Passport**
  - Search terms: `passport`, `canada passport`
  - File types: PDF, JPG, PNG  
  - Common names: `passport-scan.pdf`, `canada-passport.jpg`

- **Health Card/OHIP**
  - Search terms: `health card`, `ohip`, `health insurance`
  - File types: PDF, JPG, PNG
  - Common names: `ohip-card.jpg`, `health-card.pdf`

- **Birth Certificate**
  - Search terms: `birth certificate`, `birth cert`
  - File types: PDF, JPG, PNG

- **Citizenship Certificate**
  - Search terms: `citizenship`, `canadian citizenship`
  - File types: PDF, JPG, PNG

- **Notice of Assessment**

## Family Member Patterns

When searching for family documents, include these name variations:

- **Eric:** `eric`, `emorin`, `eric morin`
- **Tina:** `tina`, `tina park`, `christina` 
- **Jessa:** `jessa`, `jessa morin`
- **Justin:** `justin`, `justin robertson`
- **Jordan:** `jordan`, `jordan robertson`
- **Generic:** `mom`, `mother`, `dad`, `father`

## Common File Naming Patterns

### Organized Patterns
- `Category-DocumentType-Year.pdf` (e.g., `Tax-T1-2023.pdf`)
- `Year-Category-Details.pdf` (e.g., `2023-Tax-Return.pdf`)
- `DocumentType-Date.pdf` (e.g., `Passport-2024-01-15.pdf`)

### Scan Patterns
- `Scan-YYYY-MM-DD.pdf`
- `Document-Scan.pdf` 
- `IMG_XXXX.jpg` (phone camera scans)

### Download Patterns
- `filename (1).pdf` (duplicate downloads)
- `DocumentName_timestamp.pdf`
- Browser default names: `download.pdf`, `document.pdf`

## Search Strategy Tips

1. **Start broad, then narrow:** Begin with category searches, then add specific terms
2. **Include variants:** Search for both `licence` and `license`, `tax` and `return`  
3. **Use years for dated documents:** Always include year for tax returns, statements
4. **Check multiple locations:** Personal documents may be in different folder structures
5. **Consider family context:** Documents might include spouse/family member names
