# Online Research Skill

## Overview
Advanced research capabilities for gathering information from multiple online sources including Reddit, GitHub, forums, documentation sites, and technical communities.

## Capabilities
- Reddit community research
- GitHub repository analysis  
- Technical forum scraping
- Documentation site analysis
- Social media research (when accessible)
- API discovery and reverse engineering

## Usage
This skill provides research functions for:
1. **Authentication bypass techniques**
2. **API discovery and documentation**
3. **Community solutions and workarounds**
4. **Technical implementation examples**
5. **Security and automation discussions**

## Functions Available

### `research_reddit(query, subreddits=[])`
Search Reddit communities for relevant discussions

### `research_github(query, language="", stars="")`  
Search GitHub repositories and code

### `research_documentation(domain, query)`
Extract info from documentation sites

### `research_forums(query, sites=[])`
Search technical forums and communities

### `discover_apis(domain)`
Look for API endpoints and documentation

## Files
- `scripts/reddit-search.py` - Reddit research automation
- `scripts/github-search.py` - GitHub repository analysis
- `scripts/api-discovery.py` - API endpoint discovery
- `scripts/forum-crawler.py` - Forum and community research
